HTML: Cheat Sheet


Tag Name Description
<html> HTML Element All content of the html document should be contained between these tags
<head> Head Element Contains general information about an HTML page
<title> Title Element Contains a tet that defines the title of an HTML document
<h1><h2><h3><h4><h5><h6> Headings Six different sizes of heading elements
<div> Div Element Divides HTML document into sections
<ul> Unordered List ELement Creates a list in no particular order
<ol> Ordered List Element Creates list items in a sequential order
<li> List Item Element creates list items inside ordered and unordered lists
<p> Paragraph Element Contains and displays blocks of text
<span> Span Element Inline container for text that can be sued to group text for styling purposes
<em> Emphasis Element Italicize the text
<strong> Strong Element Highlights important text in bold
<br> Line Break Element Creats a line break and is used when division of text is required
<img> Image Element Embeds images into the document.
It has two requred attributes: src and alt.
<a> Anchor Element Used to create hyperlinks in a HTML document
<video> Video Element Embeds a media player into a HTML document
Semantic Tags
<header><nav><main><footer> Structual Element Creates the basic structure of the webpage
<section> Section Element Defines elements in a document with a certain theme
<article> Article Element Holds content that makes sense on its own
<aside> Aside Element Additional information that is related to the main content
<figure> Firgure Element Encapsulates all types of media
<figcaption> Figcaption Element Used to describe the media in <figure>
<video><embed><audio> Media Element Elements used for media files
Tables
<table> Table Element Contains content that is inside a two-dimensional table
<tr> Table Row Element Is used to add rows to a table
<th> Table Heading Element Is used to add titltes to rows and columns of a table
<td> Table Data Element Adds a cell of data to a table
<tbody> Table Body Element Contians all table data other than table headings and footer content
<thead> table Head Element Defines the headings of table columns
<tfoot> Table Footer Element Gives a footer content to summarize content at the end of the table
colspan Column Span (Attribute) Indicates how many colmuns the particular cell should span
rowspan Row Span (Attribute) Indicates how many rows the particular cell should span
Forms
<form> Form Element The purpose of this element is to allow users to input information and send it
<input> Input Element Too add fields for users to input information
<selet> Select Element Populated with <option> element and renders a dropdown list selection
<datalist> Datalist Element Populated with <option> elements and works with an <input> to search through choices
<textarea> Textarea Element This is a text input field that as a customizable area
When a <form> is submitted, the name of the fields that accept input and the value of those fields are sent as name=value pairs
The <form>'s action attribute determines where the forms information goes.
The <form>'s method attribute determines how the information is sent.

Notes

What is the main difference between inline styles and the <style> tag?
Inline styles directly modify individual HTML elements using a style attribute, whereas the <style> tag allows you to write CSS in a dedicated section of the HTML file. The contents within the style tag appear inside the head tag and can only contain CSS code.
